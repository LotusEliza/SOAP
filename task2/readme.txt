Задание:
1. Необходимо реализовать простой SOAP сервер "Автомагазин"
     а) сервис должен иметь wsdl, описывающий следующие функции:
     получить список автомобилей (возвращает только ID, марку и модель),
     получить детальную информацию по ID (возвращает комплекс type из 
имени модели, год выпуска, объём двигателя, цвет, макс скорость, цена),
     поиск по параметрам (в качестве параметров используется тот же 
комплекс type что и в предыдущем запросе. Поле «год выпуска» - обязательно),
     функция выполнения предварительного заказа автомобиля (ID авто, 
имя, фамилия покупателя, способ оплаты. Способ оплаты enumeration из 
"кредитная карта", "наличные")

     б) Сервис должен контролировать любые ошибки в полученном запросе и 
адекватно на них реагировать (возвращать определенную ошибку).
     Проверка должна быть на тип данных, соответствие требованиям wsdl.

     в) Необходимая информация должна находиться в базе данных.

2. Реализовать простой интерфейс + SOAP клиент для работы с сервисом.
